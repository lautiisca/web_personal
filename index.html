==== Etiquetas ====

Estos son algunos tipos de etiquetas:

= Etiqueta "h1"=

La etiqueta "h1" es utilizada para los títulos y se definió en el archivo css como color #07a3be y un tamaño de fuente de 28px

A continuación un ejemplo:

Ejemplo h1

= Etiqueta "p" =

La etiqueta "p" es utilizada para los párrafos y se definió en el archivo css como color #312609 y un tamaño de fuente de 16px

A continuación un ejemplo:

Ejemplo p

= Etiqueta "a" =

La etiquieta "a" es utilizada para agregar hipervínculos, el color se definió en archivo css con color #0077ff y un tamaño de fuente de 12px, además se colocó "text-decoration: none" para no subrayar el hipervinculo de manera automática.
